Description détaillée

Widget proposant à l'utilisateur de sélectionner un fichier.

Affiche un QLabel puis un QPushButton qui ouvre une QFileDialog. Lorsque l'utilisateur sélectionne un nouveau fichier, le QLabel affiche le chemin complet vers le fichier et le signal fileChange() est émis.
