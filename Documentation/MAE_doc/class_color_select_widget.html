Description détaillée

Widget proposant à l'utilisateur de sélectionner une couleur.

Affiche un QLabel avec la couleur par défaut, puis un QPushButton qui ouvre une QColorDialog. Lorsque l'utilisateur change la couleur, le QLabel se met automatiquement à jour avec la nouvelle couleur en fond et le signal colorChange() est émis.
