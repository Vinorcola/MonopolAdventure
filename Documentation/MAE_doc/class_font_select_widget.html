Description détaillée

Widget proposant à l'utilisateur de sélectionner une fonte de caractères.

Affiche un QLabel avec le nom de la police ainsi que des différents attributs (gras, italique, etc.), puis un QPushButton qui ouvre une QFontDialog. Lorsque l'utilisateur change la fonte de caractères, le QLabel se met automatiquement à jour avec le nouveau texte et le signal fontChange() est émis.
